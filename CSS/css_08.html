她是娱乐圈“最干净”女演员，出道21年零绯闻，38岁凭狂飙再翻红

说到战菁一，估计很多人都没有听过这个名字，因为她是典型的“戏红人不红”。

但提到她的作品，扮演的角色，大家就会恍然大悟，原来是她呀。

没错，她就是《甄嬛传》里面的流朱，那个随着甄嬛一起去到皇宫的陪嫁丫鬟，跟浣碧一起，他们组成了最强的“丫鬟联盟”，让甄嬛在那深宫后院中，感受到一丝丝温暖，只可惜，衷心的流朱死的太早，也成为了许多观众的意难平。
